# pacman

This is a terminal-based recreation of pacman in C

It use ascii for rappresent game:

	## wall
	.  ball
	o  power pill
 	() you, pacman
 	{} normal ghost
 	}{ edible ghost

it is compatible for Mac OXS, Linux and Windows
